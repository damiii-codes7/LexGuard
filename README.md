## 🚀 What is YK Legal?

YK Legal is a comprehensive AI-powered legal intelligence platform built specifically for Indian law. It brings together 8 AI tools covering everything from HR policy compliance to court judgment summarization — all free, all instant, all built on open-source technology.

Built by **Adv. Damini Yasodai** — combining deep legal knowledge with AI technology to make Indian law accessible to everyone.

## 📚 Laws Covered

**Indian Labour Laws:**
POSH Act 2013, Maternity Benefit Act 1961, Payment of Wages Act 1936, Minimum Wages Act 1948, Payment of Gratuity Act 1972, EPF Act 1952, Industrial Disputes Act 1947, Payment of Bonus Act 1965, Factories Act 1948, Code on Wages 2019

**Indian Contract & Company Laws:**
Indian Contract Act 1872, Companies Act 2013, Specific Relief Act 1963, Arbitration Act 1996, Consumer Protection Act 2019, Stamp Act 1899, Competition Act 2002, IT Act 2000, Transfer of Property Act 1882, Sale of Goods Act 1930

**Other Laws:**
IPC 1860, CrPC 1973, Constitution of India, GST Act, Income Tax Act, RTI Act 2005, Hindu Marriage Act 1955, NI Act 1881, EU AI Act

## 🚀 Run Locally

```bash
# Clone the repo
git clone https://github.com/damiii-codes7/LexGuard.git
cd LexGuard

# Create virtual environment
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Mac/Linux

# Install dependencies
pip install -r requirements.txt

# Add your Groq API key
echo GROQ_API_KEY=your_key_here > .env

# Run the app
streamlit run app.py
```
